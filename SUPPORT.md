# Support

## How To Get Help

Use the channel that best matches your question:

- usage questions: open a GitHub issue with enough context to reproduce your setup
- bug reports: use the bug report issue template
- feature requests: use the feature request template
- security issues: follow [SECURITY.md](SECURITY.md) and avoid public disclosure

## What To Include

Support requests are much easier to answer when they include:

- the `abel-edge` package version
- Python version
- operating system
- the command or import path you are using
- whether the workflow uses local CSV data, Abel price APIs, or Abel discovery
- relevant `strategies.yaml` snippets with secrets removed
- the full traceback or validation output when applicable
